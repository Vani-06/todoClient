# Hippo Tasks Frontend 🦛

The user interface for Hippo Tasks, built with React and Vite. It features an adorable, modern "Hippo" theme with a simple, yet robust, task management system.

## Tech Stack
- **React**: Library for building user interfaces
- **Vite**: Build tool for fast frontend development
- **Axios**: HTTP client for API requests
- **Lucide-React**: Icon library
- **Google Fonts**: Outfit for a rounded, adorable feel

## Layout
- **Daily Routine**: Column for routine tasks.
- **Upcoming / Next**: Column for scheduled tasks.
- **Dynamic Form**: Interactive form with conditional field selection.

## Development Features
- **Modern Styling**: Dark mode (`#121212`) with Dull Pink (`#c48b9f`) and Soft Pink (`#dfb2c1`) accents.
- **API Integration**: Connects to the backend via `http://localhost:5000/api/tasks`.
- **Micro-animations**: Smooth hover transitions and rounded card designs.
